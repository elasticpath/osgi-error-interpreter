1. Open the extensions/cortex/ext-cortex-webapp/target/ext-cortex-webapp-0.0.0-SNAPSHOT/WEB-INF folder.
  2. Within each of the subfolders, look for a JAR file similar to that is duplicated in multiple folders.
  3. Once found, check the maven-dependency-plugin execution blocks within extensions/cortex/ext-cortex-webapp/pom.xml to identify which executions are adding the bundle. Modify the configuration to prevent the bundle from being copied to one of the subfolders. Usually the bundle should be removed from the latest subfolder, from the following order:
    1. bundles-level1
    2. bundles-level2
    3. bundles-level3
    4. bundles-level4
